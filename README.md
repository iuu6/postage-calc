# 邮票计算器

简单的邮票面值计算工具，帮助用户使用最少的邮票凑齐指定金额。

## 功能

- 从 `available.json` 加载默认邮票配置
- 添加自定义邮票（存储在浏览器 Cookie）
- 计算最优邮票组合（优先凑齐金额，使用最少邮票数量）
- 支持无限张数邮票（count 设为 -1）

## 使用方法

1. 打开 `index.html`
2. 查看当前可用邮票列表
3. 可选：添加自定义邮票
4. 输入需要凑的金额
5. 点击"计算"查看最优方案

## 配置邮票

编辑 `available.json`：

```json
[
  {"value": 1.5, "count": 10},
  {"value": 1.2, "count": 5},
  {"value": 1.0, "count": -1}
]
```

- `value`: 邮票面值
- `count`: 可用张数（-1 表示无限）

## 算法

使用深度优先搜索（DFS）+ 剪枝，优先选择大面值邮票，找到使用最少邮票数量的组合。
